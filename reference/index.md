# Package index

## Metadata

- [`get_sample_info()`](https://alexslemonade.github.io/ScPCAr/reference/get_sample_info.md)
  : Get sample metadata by sample ID
- [`get_project_info()`](https://alexslemonade.github.io/ScPCAr/reference/get_project_info.md)
  : Get project metadata by project ID
- [`get_project_samples()`](https://alexslemonade.github.io/ScPCAr/reference/get_project_samples.md)
  : Get a data frame of all samples in a given project
- [`get_project_libraries()`](https://alexslemonade.github.io/ScPCAr/reference/get_project_libraries.md)
  : Get metadata for all libraries in a given project
- [`scpca_projects()`](https://alexslemonade.github.io/ScPCAr/reference/scpca_projects.md)
  : Get data frame of all ScPCA projects
